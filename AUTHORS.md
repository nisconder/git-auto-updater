## Contributors

感谢所有为 Git Auto Updater 做出贡献的人！

如果你想成为贡献者，请查看 [CONTRIBUTING.md](CONTRIBUTING.md) 了解如何参与。

## How to Add Yourself

如果你贡献了代码、文档、或其他改进，你可以将你的名字添加到这个列表：

```markdown
- nisconder (@nisconder) - Project maintainer and core development
```
