# 作者与贡献者

## 维护者

- Git Auto Updater Team

## 贡献者

感谢所有为 Git Auto Updater 做出贡献的人。

如果你想成为贡献者，请查看 [CONTRIBUTING.md](CONTRIBUTING.md)。

## 如何添加自己

如果你贡献了代码、文档或其他改进，可以将你的名字添加到下面：

```markdown
- nisconder (@nisconder) - 项目维护与核心开发
```

## 贡献类型

我们感谢以下贡献：

- 代码贡献
- 文档改进
- Bug 报告
- 功能建议
- 翻译与本地化
- 测试与质量保障
- 社区支持
